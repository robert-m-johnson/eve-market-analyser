# eve-market-analyser

## Installation

- Install MongoDB
- Run the setupDb.js script in the `scripts` directory against the DB
- Build the jar using lein: `lein uberjar`

## Usage

    $ java -jar eve-market-analyser-0.1.0-standalone.jar
